# SIB-YOLO
Lightweight RGB-infrared multimodal detector based on Structural Information Bottleneck.
The corresponding paper is under review. Full source code will be released after manuscript acceptance.

### Datasets

- **Source**: All images are from the DeMMI-RF dataset, which is selected from the following datasets:
   - [DroneVehicle](https://github.com/VisDrone/DroneVehicle)
